# فایل‌های تقویمی JSON استفاده‌شده در برنامه

این سند فقط فایل‌های JSONای را فهرست می‌کند که برنامه از مسیر `src/data/calendar-files` مستقیماً در `src/main.js` import کرده است. فایل‌های JSON دیگری که در این پوشه هستند اما در `src/main.js` import نشده‌اند، عمداً در جدول نیامده‌اند.

| فایل | کاربرد در برنامه |
| --- | --- |
| `international-marketing-occasions-simple-max.json` | مناسبت‌های بازاریابی و بین‌المللی تکمیلی |
| `global-official-gregorian-occasions-simple.json` | مناسبت‌ها و تعطیلی‌های رسمی جهانی بر پایه تقویم میلادی |
| `iran-current-persian-occasions-simple.json` | مناسبت‌های تقویم خورشیدی فعلی ایران در ساختار ساده |
| `iran-ancient-pahlavi-calendar-simple-supermax.json` | مناسبت‌های ایران باستان/پهلوی در ساختار ساده |
| `islamic-shia-occasions-simple-max.json` | مناسبت‌های اسلامی شیعه |
| `islamic-sunni-occasions-simple-max.json` | مناسبت‌های اسلامی اهل سنت |
| `islamic-shared-occasions-simple-max.json` | مناسبت‌های مشترک اسلامی |
